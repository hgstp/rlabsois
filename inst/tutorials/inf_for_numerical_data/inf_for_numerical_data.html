North Carolina births

In 2004, the state of North Carolina released a large data set containing information on births recorded in this state. This data set is useful to researchers studying the relation between habits and practices of expectant mothers and the birth of their children. We will work with a random sample of observations from this data set.

Exploratory analysis

Load the nc data set into our workspace.

download.file("http://www.openintro.org/stat/data/nc.RData", destfile = "nc.RData")
load("nc.RData")

We have observations on 13 different variables, some categorical and some numerical. The meaning of each variable is as follows.

variable description
fage father’s age in years.
mage mother’s age in years.
mature maturity status of mother.
weeks length of pregnancy in weeks.
premie whether the birth was classified as premature (premie) or full-term.
visits number of hospital visits during pregnancy.
marital whether mother is married or not married at birth.
gained weight gained by mother during pregnancy in pounds.
weight weight of the baby at birth in pounds.
lowbirthweight whether baby was classified as low birthweight (low) or not (not low).
gender gender of the baby, female or male.
habit status of the mother as a nonsmoker or a smoker.
whitemom whether mom is white or not white.

Consider the possible relationship between a mother’s smoking habit and the weight of her baby. Plotting the data is a useful first step because it helps us quickly visualize trends, identify strong associations, and develop research questions.

Exercise

Make a side-by-side boxplot of habit and weight. What does the plot highlight about the relationship between these two variables?

ggplot(nc, aes(x = habit, y = weight)) + geom_boxplot()

The box plots show how the medians of the two distributions compare, but we can also compare the means of the distributions using the group_by() function from dplyr to split the weight variable into the habit groups and then summarising the contained information by using the mean() function.

nc %>%
  group_by(habit) %>%
  summarise(mean_weight = mean(weight))

There is an observed difference, but is this difference statistically significant? In order to answer this question we will conduct a hypothesis test.

Remark: In the last example we also used the pipe operator %>%, which forwards the data from one step to the next one.

Both distributions are (moderately) skewed. But due to the large sample size, we can still apply the testing framework of Chapter 5.

Exercise

Write the hypotheses for testing if the average weights of babies born to smoking and non-smoking mothers are different.

To test the above given hypothesis, we can use the t.test() function.

t.test(weight ~ habit, alternative = "two.sided", mu = 0, data = nc)
## 
##  Welch Two Sample t-test
## 
## data:  weight by habit
## t = 2.359, df = 171.32, p-value = 0.01945
## alternative hypothesis: true difference in means is not equal to 0
## 95 percent confidence interval:
##  0.05151165 0.57957328
## sample estimates:
## mean in group nonsmoker    mean in group smoker 
##                7.144273                6.828730

Let’s pause for a moment to go through the arguments of this custom function. The first argument uses the formula notation weight ~ habit, which specifies that we are interested in differences in mean weight due to habit. The second argument alternative can be "less", "greater", or "two.sided". In our case we have two sided test, which is actually the default case. So, we could have skipped this argument. This is also true for mu, since 0 is the default value. The last argument specifies the data object, which has to contain (in our case) the variables weight and habit.
